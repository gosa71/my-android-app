# Семейный чат (Family Chat)

Обычный семейный мессенджер на Android (Kotlin + Jetpack Compose).
Переработан из прежнего «Семейного Очага»: убрана «плюшевая» тема, общая стена, задачи, календарь и радар локаций.

## Что внутри

- **Вход** — выбор профиля члена семьи (Мама, Папа, Сын, Дочь, Бабушка, Дедушка).
- **Список чатов** — общий чат семьи + отдельный личный чат с каждым членом семьи, с превью последнего сообщения.
- **Чат** — пузыри сообщений (свои справа, чужие слева), время, в групповом чате — имена отправителей.
- **Хранение** — локально в Room (офлайн). Опциональная синхронизация через Firebase Realtime Database (REST).

## Архитектура

- `data/Family.kt` — роли семьи и формирование `conversationId` (1:1 и групповой чат).
- `data/FamilyDatabase.kt` — Room: `MessageEntity` с `conversationId`.
- `data/FirebaseService.kt` — синхронизация сообщений по веткам `messages/{conversationId}/{messageId}`.
- `ui/MainViewModel.kt` — идентичность, список чатов, активная переписка, отправка и поллинг.
- `MainActivity.kt` — экраны: вход → список чатов → чат.
- `ui/theme/` — нейтральная тема (синий акцент + светлая/тёмная).

## Сборка

1. Откройте проект в Android Studio («Open» → эта папка).
2. Создайте файл `.env` (см. `.env.example`) — плагин secrets требует его наличия, хотя само приложение ключи не использует.
3. Для отладочной сборки запустите `Run` (debug). Для release — задайте переменные окружения `KEYSTORE_PATH`, `STORE_PASSWORD`, `KEY_PASSWORD` или поменяйте `signingConfig` в `app/build.gradle.kts`.

## Настоящая переписка между устройствами

По умолчанию в `MainViewModel.firebaseDatabaseUrl` указан демо-URL Firebase. **Это общая открытая база без авторизации** — для реального использования создайте свой проект Firebase, включите правила доступа и подставьте свой URL. Чтобы остаться полностью офлайн, оставьте URL пустым.
